# 🌦️ Skyvera Weather App

Skyvera Weather App is a **Full Stack Weather Application** that allows users to search and explore real-time weather data for cities across the world.

The application provides current weather conditions, hourly forecasts, upcoming weather predictions, and weather comparisons between two cities.

This project demonstrates **Full Stack Development using Java, Spring Boot, React, HTML, CSS, and JavaScript** with real-time API integration.

---

# 📌 Project Overview

Weather information plays an important role in our daily lives.  
Skyvera Weather App helps users easily access weather information for any location.

Users can:

- Search weather for any city
- View real-time temperature and weather conditions
- Check hourly and upcoming weather forecasts
- Compare weather between two cities
- Explore global weather locations on an interactive map
- Learn weather safety tips through the Weather Guide

The application integrates the **OpenWeather API** to fetch real-time weather data and display it through a modern responsive interface.

---

# 🚀 Features

## 🔍 Search Weather
Users can search for any city and get live weather data.

## 📅 Upcoming Weather Forecast
Displays upcoming weather conditions including temperature, humidity, and wind speed.

## ⏱ Hourly Forecast
Shows weather updates for the next 24 hours.

## 🌍 Global Cities Map
Interactive map showing weather locations around the world.

## ⚔️ Weather Comparison
Users can compare weather conditions between two cities.

Example:
Mumbai vs Pune

The comparison includes:
- Temperature
- Humidity
- Wind Speed
- Comfort Level

## 📊 Analytical Weather Comparison
Graph-based visual comparison between cities.

## 🏆 Weather Battle Result
The system determines which city currently has better weather conditions.

## 🌙 Dark / Light Mode
Allows users to switch between light and dark themes.

## 📘 Weather Guide
Provides useful information about:
- Heat Index
- Humidity Comfort Levels
- UV Protection
